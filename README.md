<blockquote>
<i>"The same business exists in 6 different government databases with 6 different names, addresses, and IDs. Nobody knows which one is real."</i>
</blockquote>

```bash
# Clone the repository
git clone https://github.com/Nitinmall-1390/Infralink.git
cd Infralink

# Install frontend dependencies
npm install

# Install backend dependencies
cd backend2/backend2/entitynet-backend
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env with your database credentials

# Launch both services
npm run dev          # Frontend → http://localhost:5173
python run.py        # Backend  → http://localhost:8000
```

**Environment Variables**

```env
# Database
DATABASE_URL=postgresql://user:pass@localhost:5432/infralink

# Graph Database
NEO4J_URI=bolt://localhost:7687
NEO4J_USER=neo4j
NEO4J_PASSWORD=your_password

# AI Models
SIAMESE_MODEL_PATH=./models/entity_resolver.pt
```
